## 🚀 Projects I've Built (And Why)

### 💉 VaxTrace
> *"Your vaccination records shouldn't live in a paper booklet that you lose in 3 years."*

A full-stack platform for tracking vaccination records across a lifetime. Because health data should be accessible, structured, and not dependent on whether you remember where you put that paper card.

`FastAPI` · `PostgreSQL` · `React.js` · `SQLAlchemy`

---

### 🧾 Bill Bounty *(Hackathon Project)*
> *"India loses billions every year to GST tax evasion. What if customers became the enforcement layer?"*

Built at a hackathon to tackle **GST Tax Evasion** — an incentive system where customers are rewarded for reporting unaccounted transactions. Didn't win. Learned everything.

Lesson learned: *A good problem + a bad pitch = a great experience.*

---

### 🛒 E-Commerce Backend
> *"Everyone's first real project. Mine taught me more than any tutorial ever could."*

REST API with product listings, cart management, order flow. Where I learned what "real backend work" actually feels like.

`FastAPI` · `PostgreSQL` · `SQLAlchemy` · `Pydantic`

---

### 🔐 Session-Based Auth System
> *"I wanted to understand auth — not copy-paste it."*

Built a complete session-based authentication system from the ground up. Because you don't really understand something until you've broken it yourself five times.
